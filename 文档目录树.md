## 文件作用详解

### 根目录文件

- **.gitignore** : 指定Git版本控制系统忽略的文件和目录模式，如node_modules、venv等
- **README.md** : 项目说明文档，包含项目背景信息、技术栈和团队成员介绍
- **文档目录树.md** : 项目目录结构说明文档，详细介绍每个文件的作用
- **.vscode/settings.json** : VS Code编辑器的项目特定设置，如代码格式化、语法检查等
- **.idea/** : JetBrains IDE的项目配置文件夹，包含项目设置、模块配置等

### 后端目录 (backend/)

- **APP.py** : Flask应用主文件，包含：
  - 应用初始化和配置
  - 数据库模型定义（用户、商品、销售记录）
  - API路由实现（登录、注册、商品管理、销售记录）
  - 数据库连接和初始化
- **requirements.txt** : Python依赖包列表，包含Flask及其相关扩展
- **run_backend.py** : 后端启动脚本，自动安装依赖并启动Flask服务器
- **supermarket.db** : SQLite数据库文件，存储应用数据

### 前端目录 (frontend/)

- **index.html** : Vue应用的HTML模板文件，Vite构建工具会自动注入打包后的JavaScript和CSS资源
- **package.json** : 前端项目配置文件，定义项目元数据、依赖项和脚本命令
- **package-lock.json** : 记录项目依赖的确切版本，确保在不同环境中安装相同的依赖版本
- **vite.config.js** : Vite构建工具配置文件，设置插件、路径别名和代理服务器等

#### 前端源代码 (frontend/src/)

- **main.js** : 前端应用入口文件，创建Vue实例并挂载到DOM上，同时引入Pinia状态管理和Vue Router路由
- **App.vue** : 应用的根组件，只渲染当前路由对应的视图组件

##### 路由管理 (frontend/src/router/)

- **index.js** : 配置Vue Router路由规则，定义了登录页、仪表盘和商品管理页面的路由

##### 状态管理 (frontend/src/stores/)

- **index.js** : 导出Pinia的defineStore函数，用于创建状态管理store

##### 视图组件 (frontend/src/views/)

- **Login.vue** : 登录页面组件，包含用户名和密码输入框，以及登录逻辑处理
- **Dashboard.vue** : 仪表盘页面组件，用户登录后访问的主界面，包含功能导航卡片
- **ProductManagement.vue** : 商品管理页面组件，包含：
  - 商品录入表单（名称、价格、库存、分类）
  - 商品列表显示
  - 商品搜索功能
  - 商品编辑和删除操作
